API 목록

1. 프로그램 목록 조회 API
-데이터베이스에서 모든 프로그램 데이터를 조회하여 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs
-요청 방식: GET
-매개변수: 없음
-응답: 전체 프로그램 목록 반환

2. 검색키워드 프로그램 조회 API
-매개변수로 받은 키워드를 활용하여 프로그램 제목에 해당 키워드가 포함되는 프로그램 데이터를 데이터베이스에서 조회하여 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs/keyword/:keyword
-요청 방식: GET
-매개변수: 카테고리명 (예: /api/programs/keyword/일본어)
-응답: 해당 키워드가 제목에 포함되는 프로그램 목록 반환

3. 온라인/오프라인 프로그램 조회 API
-매개변수로 받은 프로그램 운영 방식 정보를 활용하여 해당 방식에 따른 프로그램 데이터를 데이터베이스에서 조회하고 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs/method/:method
-요청 방식: GET
-매개변수: 메소드 (예: /api/programs/method/온라인)
-응답: 해당 메소드에 따른 프로그램 목록 반환

4. 현위치 기반 프로그램 조회 API 
-엔드포인트를 생성하고 POST 요청을 통해 사용자의 위치 정보를 받은 후, 받아온 위치 정보와 프로그램 경위도 정보를 비교하여 반경 2km(예시) 이내의 프로그램 데이터를 조회하고 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs/nearby
-요청 방식: POST
-요청 데이터: 사용자의 위치 정보 (경도, 위도)
-응답: 반경 2km 이내의 프로그램 목록 반환

5. 인기순 프로그램 조회 API
-데이터베이스에서 프로그램을 인기순(신청인수 기준)을 정렬하여 조회하고, 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs/popular
-요청 방식: GET
-매개변수: 없음
-응답: 인기순으로 정렬된 프로그램 목록 반환

6. 프로그램 상세 정보 조회 API
-매개변수로 받은 프로그램 ID를 활용하여 해당 프로그램의 상세정보를 데이터베이스에서 조회하여 조회한 데이터를 JSON 형식으로 응답하는 API
-엔드포인트: /api/programs/:programId
-요청 방식: GET
-매개변수: 프로그램 ID (예: /api/programs/123)
-응답: 선택한 프로그램의 상세 정보 반환

7. 프로그램 신청 API
-매개변수로 받은 프로그램 ID를 활용하여 해당 프로그램에 대한 신청을 처리하는 API로 요청 데이터를 받아 신청인수를 데이터베이스에서 1 증가시키고 성공여부를 응답해주는 API
-엔드포인트: /api/programs/apply/:programId
-요청 방식: POST
-매개변수: 프로그램 ID (예: /api/programs/123)
-요청 데이터: 사용자 정보 및 추가 필요한 데이터
-응답: 신청 결과 반환
